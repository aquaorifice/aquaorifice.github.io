Anwesha Saha

PhD Student | Database Systems Researcher



I am a PhD student at Boston University researching database systems and data management. My work involves designing practical, data-aware structures that exploit the structure of data and it's distribution to reduce latency and memory footprint. My overarching goal is to build data structures that can be integrated into practical applications and make existing technologies more practically implementable.
